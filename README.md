## Getting started

### Necesități: // Dependencies

* yolov5 requirements.txt
* librarii: pandas, openCV, matplotlib
* pytorch


### Rulare // RUN

run detect.py

hit "q" key for exiting the app
hit "enter" key for typing a letter on the screen
